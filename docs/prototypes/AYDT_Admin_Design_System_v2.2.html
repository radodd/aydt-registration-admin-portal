01
Overview
The admin portal uses a warm, professional aesthetic with a dark sidebar, wine red accents, and warm neutral surfaces. All admin-specific classes are prefixed .admin- and all admin CSS variables are prefixed --admin-. Shared components (badges, toasts, alerts) have no prefix.

Dark Sidebar Navigation

Deep maroon-black (#160C0A) sidebar with a wine-red active marker

Warm White Content

Page background (#F7F5F2) — never stark white

Wine Red Brand Accent

Wine red (#8E2A23) as the singular brand accent color

Warm Gray Neutrals

Consistent 0.5px borders, rounded-xl corners, subtle card shadows

Spacing & Density
One 4px-based scale, applied at two densities. Hierarchy is built from spacing contrast — tight gaps within a group, generous gaps between groups — so structure reads at a glance without extra color, borders, or weight doing the work.

Application — Accent Only
Wine is an accent, not a surface. It marks action and meaning — never neutral structure. Reserving it this way keeps the portal calm and lets the red that remains actually signal something.
Wine appears on
Primary buttons — one main action per view
Active-nav marker — thin left bar
Focus rings & text links
Destructive actions & overdue / error states
Wine never on
Table headers — use neutral #EDE9E4
Card borders or decorative top accents
Section chrome, breadcrumbs, dividers
Neutral facts — plan type, term name, counts
